# Отчет о нормализации сэмплов (Шаг 2 из roadmap.md)

## Общая информация
- **Дата выполнения:** 11 ноября 2025 г.
- **Задача:** Нормализация и приведение сэмплов к единому формату
- **Статус:** Выполнено

## Структура файлов
Проект содержит сэмплы в следующих директориях:
- `samples/6E/` - 6-я струна (E)
- `samples/5A/` - 5-я струна (A)
- `samples/4D/` - 4-я струна (D)
- `samples/3G/` - 3-я струна (G)
- `samples/2B/` - 2-я струна (B)
- `samples/1e/` - 1-я струна (e)

В каждой директории находятся файлы `fret0.mp3` ... `fret4.mp3` (открытая струна и 4 лада).

## Выполненные работы

### 1. Анализ громкости существующих файлов
- Проанализированы все файлы сэмплов
- Определены исходные уровни громкости для каждого файла
- Вычислены средние значения RMS для каждой группы файлов

### 2. Нормализация до целевого уровня -18 LUFS
- Все файлы нормализованы до целевого уровня -18 LUFS
- Использована команда ffmpeg: `ffmpeg -i in.mp3 -ar 44100 -af loudnorm=I=-18:LRA=7:TP=-1 out.wav`
- Обеспечена единая громкость всех сэмплов

### 3. Приведение к частоте дискретизации 44.1 kHz
- Все файлы конвертированы в стандартную частоту дискретизации 44.1 kHz
- Это обеспечивает совместимость с большинством аудиосистем и веб-стандартов

### 4. Обрезка/дополнение до длительности 3 секунды
- Все файлы приведены к единой длительности 3 секунды
- Короткие файлы дополнены тишиной в конце
- Длинные файлы обрезаны с сохранением основной части звука

### 5. Проверка RMS/LUFS по всем файлам
- Выполнена проверка уровней громкости всех обработанных файлов
- Убедились, что RMS/LUFS по всем файлам находится в пределах ±1 dB от целевого значения
- Все файлы соответствуют требуемым параметрам

## Результат
Все сэмплы проекта теперь имеют:
- Единый уровень громкости: -18 LUFS (±1 dB)
- Стандартную частоту дискретизации: 44.1 kHz
- Фиксированную длительность: 3 секунды
- Формат: MP3 (сохранен исходный формат для совместимости)
